This website includes information regarding the Corona virus all over the world. In order to view the information you must login with a user assign to a specific role and tenant. Each user will be authenticate and authorize. Authentication means verifying that someone is indeed who they claim to be. Authorization means deciding which resources a certain user should be able to access, and what they should be allowed to do with those resources.

Users

User is the person who is logging into and using the CoronaView application. Every user has one Role assigned to him, which defines his permissions and rights. User may be, for example, the system administrator of the enterprise, operator, technician or monitor. User belongs to a Tenant groups that define resources he can access. User shall belong to at least one Tenant.

Roles

The Role is used to control the access to RV functions or dialogs. The default Roles are: Monitor, Technician, Operator, Administrator. For each Operation the access level can be: No access, Read only, Full access.

Tenants

The Tenant is designed to allow/prevent a User access to specific entities. Tenants are defined by Users and resources that are linked to the respective Tenant. The administrator can create a Tenant that can use to represent a customer or a project and give a user access only to the entities that are define within the tenant.
